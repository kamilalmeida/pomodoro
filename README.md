# Pomodoro
Desenvolvi essa aplicação para aprimorar meus conhecimentos em React.
A técnica de pomodoro consiste em ajudar ao usuário a focar por 25min em uma determinada tarefa, tendo intervalo de pausas.

- ``CSS``
